Déployer un site statique avec GitHub Pages

Étapes du déploiement :

  1. Créer un dépôt GitHub public.
  2. Ajouter les fichiers HTML, CSS, JS dans le dépôt.
  3. Aller dans les paramètres du dépôt (Settings).
  4. Cliquer sur l’onglet Pages.
  5. Choisir la branche
  6. Valider et récupérer le lien fourni.
